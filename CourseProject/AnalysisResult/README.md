AnalysisResult
==============
This folder contains following result files and a Code book. These files were produced as a part Getting and Cleaning Data course project


Files
------
* Tidy_Data dataset.txt : This first dataset contains merged data of train and test with  mean, standard deviation for each measurement  and  also labelled the variables in dataset
* Tidy_Avg_Data,txt : This second dataset contains average of each variable for each activity and subject
* CodeBook.md : This file contains information of “Tidy_Data”
